# Connection icon provenance

These icons identify Connection kinds (and a few local-shell variants) in the
connection tree, tabs, and toolbars. Most are small in-house PNG marks; the
PowerShell variants are official vector marks bundled for shell identification.

## License notes

Bundling these marks identifies the underlying tool or protocol; it does not
imply endorsement. Marks may still be protected as trademarks in some
jurisdictions. Keep this folder limited to small app-identification assets.

## Sources

- `ftp.png`, `rdp.png`, `serial.png`, `ssh.png`, `telnet.png`, `terminal.png`,
  `url.png`, `vnc.png`: in-house connection-kind marks.
- `wsl.png`: in-house mark for the WSL local shell.
- `powershell5.svg`: Wikimedia Commons `File:Powershell 128.svg`, the classic
  blue Windows PowerShell (5.x) mark released by Microsoft under the MIT
  License (copyright Microsoft Corporation; trademark notice applies). Used for
  the `powershell.exe` local shell.
- `powershell.svg`: Wikimedia Commons `File:Ps black 128.svg`, the dark
  PowerShell (Core / 7) mark released by Microsoft under the MIT License
  (copyright Microsoft Corporation; trademark notice applies). Used for the
  `pwsh.exe` local shell.
