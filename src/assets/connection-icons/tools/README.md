# Coding tool and terminal logos

These SVGs back the `brand:` entries in the shared Connection icon picker.
Install Helper assets are reused directly where KKTerm already bundles the
same product logo.

- Source for SVGs in this folder: [Simple Icons](https://github.com/simple-icons/simple-icons),
  released under **CC0 1.0**.
- Files were recolored from upstream monochrome paths by applying a root `fill`
  so they remain legible in KKTerm's dark UI.
- Reused Install Helper assets keep their provenance in
  `src/assets/installer-icons/README.md`.
