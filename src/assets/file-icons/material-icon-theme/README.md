# Material Icon Theme file icons

These SVGs and `manifest.json` are vendored from `material-icon-theme@5.35.0`,
the npm package for `material-extensions/vscode-material-icon-theme`.

They are bundled for KKTerm's shared SFTP/FTP/File Explorer browser surface.
The upstream package is MIT licensed; keep `LICENSE.txt` with this directory
when updating the vendored files.
