# Install Helper icon provenance

![Downloads](https://img.shields.io/github/downloads/ryantsai/KKTerm/total)

These icons are bundled only for Install Helper catalog tiles. Prefer SVGs
from Simple Icons when available; otherwise use the smallest official or
project-repository asset that remains legible at 40-46px.

## License notes

Simple Icons itself is CC0, but its disclaimer says individual brand icons can
have their own copyright, license, trademark, and brand-guideline constraints.
Bundling these marks identifies third-party tools; it does not imply
endorsement. Recheck upstream brand guidance before using them in marketing or
larger promotional surfaces.

Icons from product repositories or official websites inherit the upstream
project/site terms and may still be protected as trademarks. Keep this folder
limited to small app-identification assets.
