# settings.fileExplorerOpenMode

- **English value**: `Open files`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/WorkspaceSettings.tsx`
- **UI role**: `label`
- **User flow**: The user sees this label above the File Explorer open-mode selector in Settings -> Workspace.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: "Open" means choosing how a local file is launched from KKTerm's File Explorer, not starting a Connection or opening a folder.
- **Domain notes**: File Explorer is the `localFiles` Connection browser; the setting controls local files only.
