# screenshots.captureFullscreen

- **English value**: `Capture full screen`
- **Namespace**: `screenshots`
- **File/component**: `src/modules/screenshots/ScreenshotsPage.tsx`
- **UI role**: `button`
- **User flow**: Header button, tray menu item, and Settings shortcut row label that captures all monitors at once.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: 'Full screen' = the entire virtual desktop across every monitor.
