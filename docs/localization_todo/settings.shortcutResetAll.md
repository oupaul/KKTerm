# settings.shortcutResetAll

- **English value**: `Reset all to defaults`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/ShortcutsSettings.tsx`
- **UI role**: `button`
- **User flow**: The user activates this button in Settings → Shortcuts to restore every shortcut binding to its catalog default before saving Settings.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: “Reset” means discard all user shortcut overrides and restore the complete default shortcut catalog, not reset one row or reset all application settings.
- **Domain notes**: “Shortcuts” means keyboard shortcuts. Unbound catalog entries remain unbound by default.
