# settings.fileExplorer

- **English value**: `File Explorer`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/WorkspaceSettings.tsx`
- **UI role**: `heading`
- **User flow**: The user sees this fieldset heading in Settings -> Workspace when configuring local File Explorer behavior.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: File Explorer is KKTerm's local filesystem browser Connection kind (`localFiles`), distinct from the single-file File Viewer Connection.
- **Domain notes**: Preserve "File Explorer" as the local browser feature name; do not translate it as a generic operating-system file manager unless the locale already does so consistently.
