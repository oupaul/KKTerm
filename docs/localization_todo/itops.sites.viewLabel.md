# itops.sites.viewLabel

- **English value**: `Site view mode`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/SitesTab.tsx`
- **UI role**: `label`
- **User flow**: Accessible group label (aria-label) for the Site View segmented control in the IT Ops drill toolbar that switches between the Server Room overview, Batch Runs, and Automations.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: "Site" is the IT Ops top-level container (a named selection of Connections), not a website. "View mode" = which of the three surfaces the right pane shows. Parallel to `itops.floorPlan.viewLabel` ("Room view mode") — keep the wording pattern consistent with that key.
- **Domain notes**: Translate "Site" the same way as the rest of the `itops` namespace (e.g. `itops.sites.heading`). Not user-visible text; screen-reader only.
