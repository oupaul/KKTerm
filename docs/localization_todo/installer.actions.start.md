# installer.actions.start

- **English value**: `Start`
- **Namespace**: `installer`
- **File/component**: `src/modules/installer/InstallerToolDialog.tsx`
- **UI role**: `button`
- **User flow**: Shown in a managed web application's Run dialog when its local runtime or registered Windows service is stopped.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: Start the installed application's managed local runtime or Windows service. This is a lifecycle action, not the general Run/execute action used for tools and commands.
- **Domain notes**: Web UI and Windows service are technical terms; keep the distinction between starting an existing service and registering a service.
