# ai.workPlanTitle

- **English value**: `Plan`
- **Namespace**: `ai`
- **File/component**: `src/ai/AssistantWorkPanel.tsx`
- **UI role**: `heading`
- **User flow**: When the AI Assistant publishes a work plan for a multi-step task (via its update_plan tool), the expanded assistant work panel shows this heading above the plan's step list.
- **Tone**: concise/neutral, single noun
- **Placeholders**: none
- **Context/meaning**: "Plan" = the assistant's intended sequence of work steps for the current request (a to-do outline), not a subscription plan, floor plan, or scheduling feature.
- **Domain notes**: Step labels under this heading are model-generated content in the user's language and are not translated through i18n; only this heading is.
