# installer.launcher.commandReference

- **English value**: `Command-line parameter reference`
- **Namespace**: `installer`
- **File/component**: `src/modules/installer/InstallerToolDialog.tsx`
- **UI role**: `link label`
- **User flow**: Shown in a coding-agent Run dialog when an official CLI parameter reference is available. Opens that vendor's documentation in the system browser.
- **Tone**: concise/neutral
- **Placeholders**: none
