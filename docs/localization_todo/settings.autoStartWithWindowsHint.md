# settings.autoStartWithWindowsHint

- **English value**: `Launches KKTerm automatically when you sign in to Windows and starts it minimized. If tray behavior is enabled, the window hides to the system tray.`
- **Namespace**: `settings`
- **File/component**: `src/settings/GeneralSettings.tsx`
- **UI role**: `fragment`
- **User flow**: The user sees this helper text under the Start with Windows minimized toggle in Settings → General → Workspace access. It explains the Windows sign-in behavior and how it interacts with tray behavior.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Domain notes**: KKTerm is the product name. Windows is the operating system name and should stay recognizable. "Tray" means the Windows system tray/notification area.
