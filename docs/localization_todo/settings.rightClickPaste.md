# settings.rightClickPaste

- **English value**: `Right-click pastes the clipboard`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/TerminalSettings.tsx`
- **UI role**: `label`
- **User flow**: Toggle in Settings → Terminal → Clipboard and paste. When enabled, right-clicking a terminal Pane pastes the clipboard directly (PuTTY-style) instead of opening the context menu.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: "Paste" = insert the system clipboard contents into the terminal input; "right-click" = the secondary mouse button.
- **Domain notes**: Behavior mirrors PuTTY's right-click paste convention; Shift+right-click still opens the context menu.
