# itops.floorPlan.zoomReset

- **English value**: `Reset zoom to 100%`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/roomViewParts.tsx`
- **UI role**: `tooltip`
- **User flow**: Tooltip on the percentage readout between the zoom stepper buttons above the Server Room floor plan and 2.5D views; clicking it returns the view to the default magnification.
- **Tone**: concise/neutral UI label
- **Placeholders**: none — "100%" is a fixed value, part of the sentence in every locale.
- **Context/meaning**: Return the room view to its default 1:1 magnification.
- **Domain notes**: Server Room View floor plan / 2.5D room. Best-effort translations for all 14 locales were added in the same change and need review.
