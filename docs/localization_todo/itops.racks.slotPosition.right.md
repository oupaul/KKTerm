# itops.racks.slotPosition.right

- **English value**: `Right`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/RackItemDialog.tsx`
- **UI role**: `label`
- **User flow**: Seen in the IT Ops Rack Device dialog after picking a half or quarter device width: chooses which horizontal slot of the U row the device occupies.
- **Tone**: concise/neutral form label
- **Placeholders**: none
- **Context/meaning**: Select option: the rightmost horizontal slot of the U row (used for both half and quarter widths).
- **Domain notes**: IT Ops Rack View domain: "U" is a rack unit row; a fractional-width device (e.g. a modem) shares one U row side by side with others. "Left/right" are horizontal positions across the rack face viewed from the front.
