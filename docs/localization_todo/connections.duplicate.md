# connections.duplicate

- **English value**: `Duplicate`
- **Namespace**: `connections`
- **File/component**: `src/modules/workspace/connections/ConnectionSidebar.tsx`
- **UI role**: label
- **User flow**: The user sees this action in a saved Connection's right-click menu. It opens an editable duplicate draft and creates the copy only after Save.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: Create a new independent saved Connection from an existing Connection's configuration, not copy text or files.
- **Domain notes**: Connection means a durable openable resource. The duplicate receives the next available `#N` name suffix and does not copy secret values.
