# dashboard.notesFont

- **English value**: `Font`
- **Namespace**: `dashboard`
- **File/component**: `src/dashboard/widgets/NotesBody.tsx`
- **UI role**: label (aria-label and tooltip for the font `<select>`)
- **User flow**: Read by screen readers when focusing the font dropdown, and shown as the native tooltip on hover. The select itself is visible in the Notes widget toolbar.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Domain notes**: Refers to the typeface used to render the note's text (Handwriting, Marker, System, Serif, Mono). Not "font size".
