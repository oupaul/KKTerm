# settings.updateOpenDownloadPage

- **English value**: `Open download page`
- **Namespace**: `settings`
- **File/component**: `src/app/AppUpdatePrompt.tsx`
- **UI role**: `button`
- **User flow**: Shown in the "Update available" dialog when a newer KKTerm release is detected on GitHub. Clicking the button opens the GitHub release page in the user's default browser so they can manually download and install the new version.
- **Tone**: concise/neutral, imperative call-to-action
- **Placeholders**: none
- **Domain notes**: Replaces the prior in-app installer flow. The button does not install anything — it just opens the release page. "Download page" refers to the GitHub Releases page for KKTerm.
