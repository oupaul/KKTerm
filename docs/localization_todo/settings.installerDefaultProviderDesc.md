# settings.installerDefaultProviderDesc

- **English value**: `Which package provider is selected by default for new installs when a tool supports both WinGet and Chocolatey. Tools that only support one provider still use their available provider.`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/InstallerSettings.tsx`
- **UI role**: `fragment`
- **User flow**: Users see this as helper text under the Settings → Install Helper default provider selector.
- **Tone**: Direct setup guidance.
- **Placeholders**: none
- **Context/meaning**: Explains that this preference only chooses the default selection for new installs and does not disable availability fallback.
- **Domain notes**: Preserve Install Helper, WinGet, and Chocolatey as product/provider names.
