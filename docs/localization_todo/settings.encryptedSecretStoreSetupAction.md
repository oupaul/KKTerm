# settings.encryptedSecretStoreSetupAction

- **English value**: `Set up`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/CredentialsSettings.tsx`
- **UI role**: `button`
- **User flow**: Button shown in Credentials Settings when encrypted file storage is selected but unavailable, so the user can reopen the setup dialog after dismissing it or after a failed launch setup.
- **Tone**: short command
- **Placeholders**: none
- **Context/meaning**: Set up means open the encrypted file storage setup dialog, not configure all Settings.
- **Domain notes**: Preserve KKTerm, Settings, Linux, password, secret, and encrypted file storage terminology where those terms are normally kept in English for the locale.
