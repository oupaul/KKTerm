# connections.ftpProtocolSftp

- **English value**: `SFTP (over SSH)`
- **Namespace**: `connections`
- **File/component**: `src/connections/ConnectionSidebar.tsx`
- **UI role**: `label`
- **User flow**: Option in the Protocol dropdown of the FTP options fieldset. Selecting this picks SFTP — a file transfer subsystem of SSH, wire-incompatible with plain FTP and FTPS.
- **Tone**: concise/neutral.
- **Placeholders**: none
- **Domain notes**: "SFTP" and "SSH" stay English. SFTP is not "Secure FTP" — it is the SSH File Transfer Protocol and runs on the SSH transport. Do not translate to wording that conflates SFTP with FTPS.
