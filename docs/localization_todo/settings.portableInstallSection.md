# settings.portableInstallSection

- **English value**: `Portable Install`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/GeneralSettings.tsx`
- **UI role**: `heading`
- **User flow**: Shown as the General Settings subsection title above the action that creates a portable KKTerm copy on installed Windows builds.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: Names the Settings area for creating a portable installation; it is not the portable-copy creation dialog title.
- **Domain notes**: “Portable” means a self-contained KKTerm copy launched from a writable local or removable folder.
