# git.branchCreated

- **English value**: `Created branch {{name}}`
- **Namespace**: `git`
- **File/component**: `src/modules/git/` (Git Browser overlay)
- **UI role**: `status`
- **User flow**: Shown in the Git Browser overlay opened from a local terminal pane or the File Explorer toolbar when the current directory is inside a git repository.
- **Tone**: concise/neutral
- **Placeholders**: `{{name}}`
- **Context/meaning**: Git version-control terminology. "commit", "branch", "merge", "stash", "fetch/pull/push", "tag", "HEAD", "SHA" are standard git terms; keep them in their established localized git form (often English in technical locales). "Stage"/"Unstage" refer to the git index.
- **Domain notes**: Git Browser is a KKTerm overlay, not a Connection or Session. SHA, HEAD, and remote/branch names stay literal.
