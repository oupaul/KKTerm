# app.trayExit

- **English value**: `Exit`
- **Namespace**: `app`
- **File/component**: `src/app/trayMenu.ts` (pushed to the native tray menu via the `update_tray_menu` command)
- **UI role**: `label`

- **Placeholders**: none
