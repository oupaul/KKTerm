# installer.launcher.title

- **English value**: `Run {{name}}`
- **Namespace**: `installer`
- **File/component**: `src/modules/installer/InstallerToolDialog.tsx`
- **UI role**: `heading`
- **User flow**: Dialog title of the mini launcher that opens when the user clicks Run on an installed command-line tool's tile in the Install Helper.
- **Tone**: concise/neutral
- **Placeholders**: `{{name}}` — the tool's product name (e.g. "Git"); must survive unchanged in every locale.
- **Context/meaning**: "Run" = start/execute the named program, not a media Play action and not a theatrical run.
- **Domain notes**: Install Helper is a top-level Module. Tool names stay untranslated.
