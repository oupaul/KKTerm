# installer.launcher.openTerminal

- **English value**: `Open terminal`
- **Namespace**: `installer`
- **File/component**: `src/modules/installer/InstallerToolDialog.tsx`
- **UI role**: `button`
- **User flow**: Primary action of the mini launcher dialog (and of the Coreutils quick-launch list) that opens a normal PowerShell window with the tool on PATH and a prefilled starter command.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: Opens a NON-elevated terminal window; distinct from `installer.quickLaunch.openTerminal`, which opens an elevated (administrator) PowerShell for Sysinternals.
- **Domain notes**: "Terminal" is a generic console window, not a KKTerm workspace Tab. PowerShell stays untranslated.
