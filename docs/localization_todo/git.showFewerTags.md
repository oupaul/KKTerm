# git.showFewerTags

- **English value**: `Show fewer`
- **Namespace**: `git`
- **File/component**: `src/modules/git/GitSidebar.tsx`
- **UI role**: `button`
- **User flow**: Git Browser sidebar Tags section. After expanding the full tag list, this clickable row collapses it back to the newest 10 tags.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: "fewer" = collapse the expanded tag list back to the preview count. Pairs with `git.showMoreTags` which expands it.
- **Domain notes**: Git terms (branch, tag, worktree, rebase, reset, stash, HEAD, remote) follow each locale's established git terminology.
