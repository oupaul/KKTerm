# settings.includesSsh

- **English value**: `Includes SSH`
- **Namespace**: `settings`
- **File/component**: `src/modules/settings/TerminalSettings.tsx`
- **UI role**: scope label
- **User flow**: Shown as a compact marker beside Terminal setting groups and rows whose behavior also applies to SSH terminal Sessions.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: Indicates that the marked Terminal setting is shared with SSH; it does not mean the setting is exclusive to SSH.
- **Domain notes**: SSH remains the technical protocol name. Terminal and Session use KKTerm domain meanings.
