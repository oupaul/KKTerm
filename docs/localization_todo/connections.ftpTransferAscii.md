# connections.ftpTransferAscii

- **English value**: `ASCII`
- **Namespace**: `connections`
- **File/component**: `src/connections/ConnectionSidebar.tsx`
- **UI role**: `label`
- **User flow**: Option in the Transfer-type dropdown. Selecting ASCII converts line endings between local and remote conventions (FTP `TYPE A`); usually unsafe for non-text files.
- **Tone**: concise/neutral.
- **Placeholders**: none
- **Domain notes**: "ASCII" is the technical term and stays English/uppercase. Translators should not localize it (parallel to how "UTF-8" stays English elsewhere).
