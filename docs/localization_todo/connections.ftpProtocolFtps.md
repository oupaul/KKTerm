# connections.ftpProtocolFtps

- **English value**: `FTPS (TLS)`
- **Namespace**: `connections`
- **File/component**: `src/connections/ConnectionSidebar.tsx`
- **UI role**: `label`
- **User flow**: Option in the Protocol dropdown of the FTP options fieldset. Selecting this picks TLS-encrypted FTP per RFC 4217 and reveals the TLS mode (explicit vs implicit) sub-selector.
- **Tone**: concise/neutral.
- **Placeholders**: none
- **Domain notes**: "FTPS" and "TLS" are technical acronyms that stay English. FTPS specifically means FTP over TLS (RFC 4217), not SFTP. Translators should not expand the acronym or merge it with SFTP.
