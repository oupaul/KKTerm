# connections.ftpOptions

- **English value**: `FTP options`
- **Namespace**: `connections`
- **File/component**: `src/connections/ConnectionSidebar.tsx`
- **UI role**: `heading`
- **User flow**: Legend on the fieldset grouping protocol / mode / TLS / transfer-type / UTF-8 / hidden-files / cert-bypass controls inside the Add Connection and Edit Connection dialogs when the user is creating or editing an FTP Connection. Parallel to the existing "RDP options" and "VNC options" legends.
- **Tone**: concise/neutral, mirrors existing `connections.rdpOptions` and `connections.vncOptions` legends.
- **Placeholders**: none
- **Domain notes**: FTP here is the umbrella Connection kind that covers plain FTP, FTPS, and SFTP sub-protocols, not just plain FTP. Keep "FTP" as the English acronym.
