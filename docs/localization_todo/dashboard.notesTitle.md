# dashboard.notesTitle

- **English value**: `Notes`
- **Namespace**: `dashboard`
- **File/component**: `src/dashboard/registry/builtInRegistry.ts` (resolved via `t(titleKey)` in `src/dashboard/edit/CatalogOverlay.tsx` and `src/dashboard/view/WidgetFrame.tsx`)
- **UI role**: heading
- **User flow**: Shown as the widget's title in the Dashboard catalog, in the Ambient preset title strip on the placed widget, and in the Customize popover preview. The user sees it whenever they browse for or interact with the Notes widget.
- **Tone**: concise/neutral, one-word noun
- **Placeholders**: none
- **Domain notes**: The widget mimics a paper post-it note. Use a generic word for "Notes" (plural collection of short reminders) rather than "Memo", "Note", or "Sticky Note" if the locale has a clear standard for the desktop-widget meaning. Avoid product names like "Sticky" / "Stickies".
