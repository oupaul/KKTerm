# connections.ftp

- **English value**: `FTP`
- **Namespace**: `connections`
- **File/component**: `src/connections/ConnectionMenus.tsx`
- **UI role**: `label`
- **User flow**: The user opens the "Add connection" menu in the Connections panel and sees a list of Connection kinds to pick from. "FTP" is the entry that creates a first-class File Transfer Connection covering plain FTP, FTPS, and SFTP sub-protocols.
- **Tone**: concise/neutral, matches the other kind labels (SSH, Telnet, RDP, VNC).
- **Placeholders**: none
- **Domain notes**: FTP is a technical term and typically stays English across locales — same convention as SSH, SFTP, RDP, VNC, Telnet. The FTP Connection kind also handles SFTP and FTPS sub-protocols; localizers should not translate it into "File Transfer" or expand the acronym.
