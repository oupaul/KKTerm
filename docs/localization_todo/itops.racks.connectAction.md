# itops.racks.connectAction

- **English value**: `Connect`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/RackItemConnectPopover.tsx`
- **UI role**: `button`
- **User flow**: Shown at the right edge of each Connection row in the rack-device connect popover (click a placed rack device with bound Connections). Clicking the row opens that Connection's Session.
- **Tone**: concise action verb
- **Placeholders**: none
- **Context/meaning**: "Connect" = initiate/open the network Session for this Connection (verb, imperative). Not a noun and not "combine/link".
- **Domain notes**: "Connection" is the KKTerm durable openable resource; "Session" is the live runtime state. Best-effort translations for all 13 locales were added in the same change — review them.
