# itops.editor.siteLabel

- **English value**: `Site`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/AutomationEditor.tsx`
- **UI role**: `label`
- **User flow**: Accessible label (aria-label + tooltip) of the Site select in the Automation editor header, next to the rule's name. It sets the Automation's durable Site binding — which Site's Automations segment lists the rule.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: "Site" is the IT Ops top-level container (a named selection of Connections), not a website. Singular — one binding per Automation. Keep the term identical to `itops.batchRuns.siteLabel`.
- **Domain notes**: Translate "Site" the same way as the rest of the `itops` namespace.
