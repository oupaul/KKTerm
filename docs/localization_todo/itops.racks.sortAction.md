# itops.racks.sortAction

- **English value**: `Sort`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/SitesTab.tsx`
- **UI role**: `button`
- **User flow**: Shown on the Server Rooms tree toolbar and as the parent submenu in that row's native context menu. It opens the available name-order choices.
- **Tone**: `concise/neutral`
- **Placeholders**: `none`
- **Context/meaning**: `Sort` means reorder the Server Room rows by name, not filter them, organize room contents, or persist a Server Room field.
- **Domain notes**: `Server Room` is the durable container between Site and Rack in the Site → Server Room → Rack topology.
