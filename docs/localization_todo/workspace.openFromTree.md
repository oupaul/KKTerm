# workspace.openFromTree

- **English value**: `Open a Connection from the tree, or create a new Connection:`
- **Namespace**: `workspace`
- **File/component**: `src/modules/workspace/WorkspaceCanvas.tsx`
- **UI role**: `fragment`
- **User flow**: The user sees this guidance when the active Workspace has no open Tabs, immediately above direct links for creating each supported Connection type.
- **Tone**: concise/direct setup guidance
- **Placeholders**: none
- **Context/meaning**: “Open” starts an existing durable Connection from the tree; “create” opens the add flow for a new durable Connection.
- **Domain notes**: Capitalized “Connection” is KKTerm's durable openable resource, not a live Session or frontend Tab. Connection type names such as SSH, RDP, and VNC typically stay English.
