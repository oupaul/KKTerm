# itops.sites.viewOverview

- **Namespace**: `itops`
- **File/component**: `src/modules/itops/SitesTab.tsx`
- **UI role**: `button`
- **User flow**: First segment of the Site View segmented control in the IT Ops drill toolbar. Selecting it shows the default Site surface: the free-form Server Room card overview.
- **Tone**: concise/neutral
- **Placeholders**: none
- **Context/meaning**: "Overview" = the summary/topology surface of one Site (its Server Room cards), as opposed to the sibling "Batch Runs" and "Automations" segments (`itops.tabs.runs`, `itops.tabs.autos`). Not a document abstract.
- **Domain notes**: Keep it short — it renders inside a compact segmented button next to the Batch Runs and Automations labels.
