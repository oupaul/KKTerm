# installer.steps.installNamed

- **English value**: `Install {{name}}`
- **Namespace**: `installer`
- **File/component**: `src/modules/installer/InstallerToolDialog.tsx`
- **UI role**: `status`
- **User flow**: The active package-installation stage names the managed npm application currently being installed.
- **Tone**: `short progress phrase`
- **Placeholders**: `{{name}}`; preserve the token verbatim so translators can reorder the application name.
- **Context/meaning**: Install means add the named software to the local machine, not launch or run it.
- **Domain notes**: `{{name}}` is a catalog product name such as n8n and must not be translated by the surrounding phrase.
