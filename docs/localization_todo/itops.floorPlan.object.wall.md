- **English value**: `Partition wall`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/roomViewParts.tsx` (object picker card), `src/modules/itops/ServerRoomFloorPlan.tsx` / `src/modules/itops/ServerRoomIsoView.tsx` (object tooltip)
- **UI role**: `label`
- **User flow**: Server Room View edit mode shows the object picker column; this names the partition-wall card and the placed wall's hover tooltip in the floor plan and 2.5D layouts.

- **Context/meaning**: A room partition wall segment (painted CMU masonry) placed on the server-room floor grid; it auto-connects to walls on adjacent cells. Not a software firewall and not the outer building wall.
