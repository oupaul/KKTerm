# connections.ftpProtocolFtp

- **English value**: `Plain FTP`
- **Namespace**: `connections`
- **File/component**: `src/connections/ConnectionSidebar.tsx`
- **UI role**: `label`
- **User flow**: Option in the Protocol dropdown of the FTP options fieldset. Selecting this picks the unencrypted RFC 959 FTP transport.
- **Tone**: concise/neutral.
- **Placeholders**: none
- **Domain notes**: "FTP" stays English. "Plain" here means unencrypted (credentials in cleartext), as opposed to FTPS. Avoid translating to wording that implies "default" or "standard" — those would mislead users about the security trade-off.
