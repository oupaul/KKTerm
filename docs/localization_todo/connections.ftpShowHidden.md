# connections.ftpShowHidden

- **English value**: `Show hidden files`
- **Namespace**: `connections`
- **File/component**: `src/connections/ConnectionSidebar.tsx`
- **UI role**: `label`
- **User flow**: Checkbox label inside the FTP options fieldset. When enabled, remote entries whose names start with "." are included in directory listings.
- **Tone**: concise/neutral.
- **Placeholders**: none
- **Domain notes**: "Hidden" follows the Unix dotfile convention rather than the Windows hidden-attribute convention. Translators should use the term local Unix/Linux file managers use.
