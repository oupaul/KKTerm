# itops.racks.widthFractionLabel

- **English value**: `Device width`
- **Namespace**: `itops`
- **File/component**: `src/modules/itops/RackItemDialog.tsx`
- **UI role**: `label`
- **User flow**: Seen in the IT Ops Rack Device dialog when adding or editing a switch, router, or generic device: chooses whether the faceplate takes the full rack width or shares one U row with other devices (half or quarter width).
- **Tone**: concise/neutral form label
- **Placeholders**: none
- **Context/meaning**: Field label for the faceplate width selector (full/half/quarter). "Width" is the physical horizontal size across the rack face, not a column width.
- **Domain notes**: IT Ops Rack View domain: "U" is a rack unit row; a fractional-width device (e.g. a modem) shares one U row side by side with others. "Left/right" are horizontal positions across the rack face viewed from the front.
